## 📌 POP 생산시점관리
스마트 팩토리 환경에서 실시간으로 PCB 생산 데이터를 수집, 분석, 시각화하는 WPF 기반 데스크톱 애플리케이션입니다.  
PLC, MySQL, OpenCV, YOLO 모델을 연동하여 설비의 생산성과 품질을 통합적으로 관리할 수 있도록 구현하였습니다.

> **기간**: 2025.07.01 ~ 2025.07.30
> **인원**: 2인  
> **목표**: PCB 생산 상태의 실시간 모니터링 제어 및 품질 불량 자동 감지
